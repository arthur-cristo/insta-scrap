# Análise de Performance: Instagram (Salvador, BA)

Este projeto realiza a extração e análise de dados de perfis comerciais no Instagram para identificar padrões de engajamento, visualizações e melhores horários de postagem.

## 📊 Visão Geral da Análise
O objetivo deste script é processar dados brutos (JSON) de perfis do Instagram, classificar por categoria (Estética, Moda, Restaurante) e gerar insights acionáveis sobre a performance dos conteúdos.
